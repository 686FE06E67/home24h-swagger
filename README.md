## Features:

- ### User / Anonymous:

- Xem các bài đăng mới nhất

- Tạo mới bài đăng (yêu cầu đăng nhập)

- Tạo tài khoản mới để đăng bài và quản lý bài đăng

- ### Admin:

- Duyệt/ xóa bài đăng của người dùng

- Cấp quyền/ thu hồi quền hạn người dùng
